# Operating Systems

There are thousands of operating systems out there that are running everything from smart fridges to supercomputers, but almost all the mainstream ones trace back to just two family trees. 

- Modern Windows (including Windows 10, Windows Server, and the Xbox operating system) is built on an architecture called Windows NT. 
- The Unix family is massive and incredibly diverse. It includes macOS, iOS, Android, and the entire world of Linux. 

Because macOS and Linux share Unix roots, their command-line environments are nearly identical. A basic command written for Unix fifty years ago works the exact same way on a modern MacBook or a Linux server today. 
## Unix and History

Unix started at Bell Labs in the mid-1960s. It introduced concepts we take for granted now, like nested folders, multi-user setups. The Unix philosophy is simple: build small, single-purpose tools that do one thing well, and chain them together to solve complex problems. 

Unix is a trademark owned by a consortium called the Open Group. Operating systems that pay for and pass their expensive certification process get to call themselves true Unix systems and systems that act like Unix but skip the official certification are called Unix-like for legal purposes but both work in the same way. 
## Linux and GNU

In the 1980s, Richard Stallman launched the Free Software Movement against proprietary and closed source software He started the GNU Project to build a free, Unix-compatible operating system from scratch.

By the early 1990s, GNU had built almost every piece of their operating system except the core, the kernel. That is where Linus Torvalds came in. He wrote a free kernel, named it Linux, and combined it with the GNU tools to build a complete, working operating system. 

Linux distributions like Ubuntu and Fedora bundle the Linux kernel with GNU tools, a package manager, and a graphical desktop for end users.
